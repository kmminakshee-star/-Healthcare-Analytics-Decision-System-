# Healthcare Analytics & Decision System

## 📌 Project Description

This project is a data analytics dashboard developed using Python and Streamlit.
It helps in analyzing hospital datasets to generate meaningful insights and supports better decision-making in healthcare systems.

The system allows users to explore data visually through graphs and interactive components.

---

## 🎯 Objectives

* To analyze hospital data effectively
* To visualize trends and patterns
* To support decision-making using data insights

---

## 🚀 Features

* Interactive dashboard
* Data visualization (charts & graphs)
* Filtering options for better analysis
* User-friendly interface

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Plotly

---

## 📂 Dataset

* hospital_dataset.csv (used for analysis)

---

## ▶️ How to Run the Project

1. Install required libraries:
   pip install streamlit pandas plotly

2. Run the application:
   streamlit run app.py

---

## 📊 Output

* Interactive dashboard displaying hospital data
* Graphs showing trends and comparisons
